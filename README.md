# ESG Impact OS

A Next.js foundation for a more complete ESG and environmental impact platform.

## What this build does

- Replaces a basic workbook with a real product shell for measurement, reporting, and action.
- Seeds the app with the same kinds of inputs already present in your spreadsheet:
  - energy and emissions
  - water and waste
  - audit trail records
  - framework mappings
  - validation checks
- Adds the product layers the spreadsheet was missing:
  - facility drill-down
  - hotspot analysis
  - scenario planning
  - audit readiness scoring
  - BRSR / ESRS / GRI reporting views

## How to run

```bash
npm install
npm run dev
```

## Benchmark research used to shape the product

- Greenly highlights fast Scope 1, 2, and 3 accounting, supplier screening, factor matching, and decarbonization roadmaps:
  - https://greenly.earth/en-us/products/carbon-footprint
- Watershed emphasizes AI and API ingestion, audit-ready measurement, and multi-framework reporting:
  - https://watershed.com/platform/measure
- Credibl emphasizes broad ESG KPI coverage, AI-based assurance, and one-click reporting across frameworks:
  - https://www.crediblesg.com/
- Persefoni emphasizes controls-heavy carbon accounting, audit logs, and disclosure readiness:
  - https://www.persefoni.com/business/carbon-footprint-measurement-analytics
- GHG Protocol remains the core accounting baseline for scopes and value chain logic:
  - https://ghgprotocol.org/corporate-standard
  - https://ghgprotocol.org/corporate-value-chain-scope-3-standard
